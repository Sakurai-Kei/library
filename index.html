Welcome

The purpose of this web is to keep track of books that you read.

As user, you can input book details such as title, author, page number and read status. To input one book, please click on the add button on the bottom right.

After submitting one book, the web will show the list of book in "cards". There will be two background color for these cards. Green indicates you have read the book while red means you have not. User may change their read status simply by clicking the cards.

User may remove their entry. To do so, start by clicking the remove button on the bottom right. User will then enter book selection mode, where user can click the cards to select which books to delete. User may delete multiple book at a time. Chosen books will be highlighted on their card border. Once user have finished, simply click the check mark button on bottom right to confirm deletion.

Finally, the web supports localStorage. Upon submitting, removing or changing read status, the changes will be saved. User will not lose their record upon refreshing the web page.
